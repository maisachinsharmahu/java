# JSP Form Validation

This practical shows how to perform server-side validation using JSP scripts.

### Concepts Covered:
- **request.getParameter()**: Used to retrieve form data.
- **Data Sanitization**: Checking for empty strings and whitespace.
- **Type Conversion**: Using `Integer.parseInt()` and handling potential exceptions.
- **Conditional Logic**: Validating business rules (e.g., Age >= 18).

### How to Run:
- Deploy on a web server like Apache Tomcat.
- Open `registration.jsp` in your browser.
