# Java Beans Concept

Java Beans are reusable software components that follow specific naming conventions.

### Rules for a Java Bean:
1. It must implement `java.io.Serializable`.
2. It must have a public **no-argument constructor**.
3. All properties should be **private**.
4. Access to properties must be through **getters and setters**.

### How to Run:
1. Compile: `javac StudentBean.java BeanDemo.java`
2. Run: `java BeanDemo`
