# Interactive Applet (Event Handling)

This practical shows how to handle user interaction (Mouse Clicks) inside an Applet.

### Key Concepts:
- **MouseListener**: Interface used to capture mouse events like clicks.
- **repaint()**: Schedules the applet to be redrawn, which calls the `paint()` method.
- **init()**: The first method called in the applet lifecycle, used for initialization and adding listeners.

### How to Run:
1. Compile: `javac ShapeApplet.java`
2. Run: `appletviewer ShapeApplet.java`
