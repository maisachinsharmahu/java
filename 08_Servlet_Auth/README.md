# User Validation using Servlets

This practical demonstrates handling form submissions and validating credentials using a Java Servlet.

### Key Concepts:
- **HttpServlet**: The base class for creating Servlets.
- **doPost()**: Method called when a form is submitted with `method="POST"`.
- **HttpServletRequest**: Used to read data from the client.
- **HttpServletResponse**: Used to send HTML content back to the client.

### How to Run:
- Compile using `servlet-api.jar` in your classpath.
- Configure the servlet mapping in `web.xml` or use `@WebServlet` annotation.
