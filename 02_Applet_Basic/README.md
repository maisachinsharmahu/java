# Java Applet Programming

Applets are small Java programs that can be embedded in web pages and run in a browser or using `appletviewer`.

### Concepts Covered:
- **paint(Graphics g)**: The life cycle method used to draw text, shapes, and images.
- **Graphics Object**: Provides methods like `drawString`, `drawRect`, and `setColor`.
- **AWT Package**: Used for components and graphical output.

### How to Run:
1. Compile: `javac HelloApplet.java`
2. Run using appletviewer: `appletviewer HelloApplet.java` (using the embedded comment tag)
