# Java Technology Practical Examination (Advanced Java)

This repository contains 10 practical programs designed for the Java/Web-Tech Practical Examination.

---

### 1. [Socket Programming](./01_Socket_Programming/README.md)
Demonstrates Client-Server communication using `ServerSocket` and `Socket` classes.

### 2. [Basic Applet](./02_Applet_Basic/README.md)
Shows the Applet lifecycle and basic graphics (`paint` method) in Java.

### 3. [Multithreading](./03_Multithreading/README.md)
Running multiple tasks concurrently using the `Thread` class and `sleep()` method.

### 4. [Interactive Applet](./04_Applet_Shapes/README.md)
Handling user interaction (Mouse Clicks) inside an applet using `MouseListener`.

### 5. [Java Beans](./05_Java_Beans/README.md)
Implementing reusable components with private properties, getters/setters, and Serializable.

### 6. [JSP Database Insert](./06_JSP_Database/README.md)
Connecting JSP to MySQL using JDBC to perform an SQL INSERT operation.

### 7. [JSP Form Validation](./07_JSP_Validation/README.md)
Server-side validation of user input (Age, Name) within a JSP script.

### 8. [Servlet Authentication](./08_Servlet_Auth/README.md)
Creating a Controller Servlet to validate user credentials from a form.

### 9. [Servlet Cookies](./09_Servlet_Cookie/README.md)
Setting and managing browser cookies for state management in Servlets.

### 10. [Full Stack Java App](./10_Java_FullStack/README.md)
A small MVC application integrating Servlets, JSP, Session handling, and Model classes.
