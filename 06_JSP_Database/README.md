# JSP Database Connectivity

This practical demonstrates how to connect a JSP page to a MySQL database and perform an insert operation.

### Key Concepts:
- **JDBC**: Java Database Connectivity API.
- **DriverManager**: Used to establish a connection.
- **PreparedStatement**: Used for executing parameterized SQL queries safely.
- **executeUpdate()**: Method used for INSERT, UPDATE, or DELETE operations.

### Configuration:
- Ensure the MySQL JDBC driver is in the `lib` folder.
- Database: `viva_db`, Table: `users`.
