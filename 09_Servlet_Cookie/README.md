# Cookie Management using Servlets

This practical shows how to store small pieces of data (Cookies) on the client side using Java Servlets.

### Key Concepts:
- **Cookie Class**: Part of `javax.servlet.http` package.
- **response.addCookie()**: Method used to send the cookie header to the browser.
- **setMaxAge()**: Sets the expiration time for the cookie in seconds.
- **State Management**: Cookies help keep track of user sessions across multiple requests.

### How to Run:
- Run the servlet on a server.
- The browser will receive the cookie in the response header.
